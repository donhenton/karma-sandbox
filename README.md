# karma-sandbox

Includes these features

* gulp
* karma
* jasmine
* fixtures (with and without jquery)
* Standalone testing
* coverage with Istanbul
* html reporting 

## Tags
1.0.0 contains older code using grunt and qunit


## Branches
Mountebank-branch demonstrates running [Mountebank](http://www.mbtest.org)
in testing scenario
